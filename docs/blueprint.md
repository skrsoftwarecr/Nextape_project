# **App Name**: NEXTAPE

## Core Features:

- Sony-Style Hero Experience: Typographic landing page with asymmetric numbering (01, 02, 03) and editorial-style value propositions instead of icon cards.
- Social Developer Auth: Integration of Firebase Auth for GitHub and Google OAuth, featuring a dual-state login and registration modal with account type selection.
- Developer Metric Panel: Personalized dashboard visualizing skill scores, compatible positions, and profile traffic analytics powered by Firestore data.
- Compatibility Scoring Engine: A deterministic logic-based algorithm that calculates tech stack compatibility percentage based on developer assessment results and job requirements.
- THE LINE: Narrative Assessment: Immersive, typewriter-driven multiple-choice assessments with immediate feedback within a dark-mode cinematic interface.
- Job Feed & Advanced Filtering: Dynamic listings with modifiable keyword search and modality filters using flexible Firestore querying.
- Verified Progress Profiles: Dynamic progress bars mapping developer skills categorized by verified 'The LINE' results.

## Style Guidelines:

- Primary and background color: #FDFFFC; Secondary: #235789; Secondary accent for elements: #C1292E; Tertiary: #F1D302; Final detail elements: #020100.
- Headlines and display text: 'Playfair Display' (serif); UI elements and body text: 'DM Sans' (sans-serif).
- Minimalist editorial structure inspired by apple.com; utilizes generous white space, heavy borders instead of shadows, and no card containers or glassmorphism.
- Immersive narrative storytelling within 'The LINE' using high-contrast typewriter text transitions and instant-reactive laptop state color shifts.
- Functional hierarchy is expressed through large typography, asymmetric numbers, and structural separation borders instead of standard icon sets.
