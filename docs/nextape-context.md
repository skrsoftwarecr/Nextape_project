# NEXTAPE — Contexto del Proyecto

NEXTAPE es una plataforma de evaluación técnica y matching laboral para desarrolladores de software.

## Objetivos

1. Evaluar habilidades técnicas mediante evaluaciones de criterio profesional (THE LINE).
2. Generar Skill Scores verificables almacenados en `user_skill_scores`.
3. Evidencia opcional mediante GitHub — nunca obligatoria.
4. Generar roadmaps personalizados basados en Skill Scores.
5. Calcular compatibilidad entre candidatos y vacantes (Compatibility Engine).
6. Permitir que empresas publiquen vacantes con skills requeridas.
7. Ranking de candidatos basado en Skill Scores y evidencia.

## Los tres pilares

- **THE LINE** — Motor de evaluación: escenarios reales, respuestas predefinidas.
- **CORE** — Núcleo de identidad del developer: scores, evidencia, compatibilidad.
- **Compatibility Engine** — Motor determinístico de matching developer ↔ vacante.

## Lo que NEXTAPE no es

- No es un juez de código (no se escribe código en las evaluaciones).
- No es HackerRank (no mide memorización ni sintaxis).
- No requiere GitHub para funcionar.
